Instructions:

Click two spots in the green outfield to set catchers, then click home base to throw a pitch and swing a ball.

If the ball lands on a yellow square, you're out! If it lands in the green your batter moves forward a base.

See how many points you can score before you have three outs!

You can find the github repository, and more code (including working server pages) here: https://github.com/ergray/baseballgame.
